# Настройка отправки формы на email

## Текущая настройка

Форма настроена на отправку писем на адрес: **sashabobr2020@mail.ru**

## Способ 1: Mailto (работает сразу)

По умолчанию форма использует метод `mailto`, который открывает почтовый клиент пользователя. Это работает сразу без дополнительной настройки.

## Способ 2: EmailJS (рекомендуется для продакшена)

Для более профессиональной отправки писем без открытия почтового клиента, настройте EmailJS:

### Шаги настройки:

1. **Зарегистрируйтесь на EmailJS**
   - Перейдите на https://www.emailjs.com/
   - Создайте бесплатный аккаунт

2. **Создайте Email Service**
   - В панели управления перейдите в "Email Services"
   - Добавьте новый сервис (Gmail, Outlook, или другой)
   - Следуйте инструкциям для подключения вашего email

3. **Создайте Email Template**
   - Перейдите в "Email Templates"
   - Создайте новый шаблон
   - Используйте следующие переменные:
     - `{{to_email}}` - получатель (sashabobr2020@mail.ru)
     - `{{from_name}}` - имя отправителя
     - `{{from_phone}}` - телефон
     - `{{from_email}}` - email отправителя
     - `{{message}}` - сообщение
     - `{{subject}}` - тема письма

4. **Получите ключи доступа**
   - Public Key: в разделе "Account" → "General"
   - Service ID: в разделе "Email Services"
   - Template ID: в разделе "Email Templates"

5. **Обновите конфигурацию**
   - Откройте файл `js/email-handler.js`
   - Найдите секцию `EMAILJS_CONFIG`
   - Замените значения:
     ```javascript
     const EMAILJS_CONFIG = {
         enabled: true, // Измените на true
         publicKey: 'ваш_public_key',
         serviceID: 'ваш_service_id',
         templateID: 'ваш_template_id'
     };
     ```

6. **Проверьте работу**
   - Откройте страницу контактов
   - Заполните и отправьте форму
   - Проверьте почту sashabobr2020@mail.ru

## Пример шаблона EmailJS

```
Тема: {{subject}}

Новая заявка с сайта БИС

Имя: {{from_name}}
Телефон: {{from_phone}}
Email: {{from_email}}

Сообщение:
{{message}}

---
Это письмо отправлено автоматически с сайта БИС.
```

## Примечания

- Бесплатный план EmailJS позволяет отправлять до 200 писем в месяц
- Для больших объемов рассмотрите платные планы
- Mailto метод работает всегда, но требует наличия почтового клиента у пользователя
